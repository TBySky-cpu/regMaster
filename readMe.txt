в базе данных данные сохраняются в следующем виде
  "Month":3,"FirstDay":3,"Year":2025,
  Month месяц - 0 это январь 1 -февраль ....11 -декабрь
  FirstDay -первый день -день недели на первое число данного месяца  1 -воскресенье 2 -понедельник ...7 -суббота
  Year -год 2025 -это 2025 год (без сдвигов)
  при редактировании базы данных не забывать об этих особенностях
  Так сделано для совместимости с с самыми первыми версиями программы
